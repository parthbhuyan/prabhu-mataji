## 🪷 Features

### 🌙☀️ Light & Dark Mode
- Elegant **animated toggle button** in the navbar (sun ☀️ / moon 🌙 icons)
- **Dark Mode** — deep Krishna-blue with royal gold glows, divine particles
- **Light Mode** — warm saffron-cream inspired by sacred manuscript pages and a sunlit Vrindavan morning
- Smooth **0.5s transitions** across every element — backgrounds, cards, text, borders
- **Persists across visits** via `localStorage`

### 🎬 Cinematic Hero Section
- Animated **triple mandala rings** spinning at different speeds
- **Divine particle field** with 130+ glowing particles (color-synced to theme)
- **Floating lotus petals** drifting across the full page
- **Golden light ray** effect from above
- **Live wish carousel** cycling through all 6 Krishna-conscious anniversary wishes

### 💍 6 Sacred Anniversary Wishes
Each wish is displayed both in the hero carousel and as a **glassmorphism card**:

| # | Wish |
|---|------|
| 1 | ✨ Celebrating One Sacred Year of Krishna-Centered Love |
| 2 | 🌸 One Beautiful Year of Divine Togetherness in Krishna's Grace |
| 3 | 🪔 A Year of Sacred Union Under the Blessings of Sri Krishna |
| 4 | 💫 One Divine Year of Love, Devotion & Krishna's Blessings |
| 5 | 🌺 Celebrating One Year of Sacred Krishna-Conscious Togetherness |
| 6 | ✨ Happy 1st Anniversary of Your Krishna-Centered Sacred Marriage |

### 📸 Divine Couple Portrait
- Wedding photo in a **golden glowing frame** with corner ornaments
- **Radial aura glow** pulsing behind the image
- Smooth **zoom-in on hover**
- Gradient border using gold, lotus pink, and saffron

### 🪔 Divine Blessings Section
- Animated **diya lamps** flickering with glow effects
- Rich devotional blessings invoking **Radha-Krishna**, Yamuna, Ganga, tulsi
- **Radha-Govinda Ki Jai** badge

### 📜 Journey Timeline
- 4 animated **glassmorphism timeline cards** with scroll-reveal
- Events: Wedding Day → First Year → Devotional Journey → Growing in Love & Bhakti
- Alternating left-right layout with glowing dot connector

### 💌 Personal Message
- **Typewriter animation** — letter by letter reveal as you scroll into the section
- Heartfelt message from the Dasanudasa
- Elegant **handwritten-style signature**

### 🌿 Divine Quote Section
- Large centered **Cormorant Garamond** italic quotes
- Highlighted golden keywords
- Soft glowing orb background

### 🎉 Celebration Section
- **"01" year badge** with golden gradient number
- 5 **icon pills**: Anniversary · Sacred Love · Bhakti Life · Krishna's Grace · Kirtan Joy
- Full **Maha-mantra** in the footer

---

## 🛠️ Tech Stack

| Technology | Details |
|------------|---------|
| **HTML5** | Semantic markup, single-file architecture |
| **CSS3** | Custom properties, glassmorphism, keyframe animations, grid, clamp() |
| **Vanilla JavaScript** | Canvas API, Intersection Observer pattern, localStorage |
| **Google Fonts** | Playfair Display · Cinzel · Cormorant Garamond · Poppins |
| **Font Awesome 6** | CDN-loaded icon library |
| **Canvas API** | Particle system + falling petal animation |

**No frameworks. No dependencies. No build step required.**

## 🚀 Getting Started

### Option 1 — Open directly
```bash
# Simply double-click anniversary.html
# OR open in browser:
open anniversary.html
```

### Option 2 — Local server (optional)
```bash
# Python
python3 -m http.server 8000

# Node.js
npx serve .
```
Then visit `http://localhost:8000/anniversary.html`
